## Task

### Croatian
```
Napišite algoritam za EREW PRAM računalo
koji za zadani niz cjelobrojnih vrijednosti duljine n
uz najviše n procesora
otkriva duljinu najduljeg neprekinutog podniza elemenata s jednakom vrijednošću
(ispisati duljinu podniza).
Na raspolaganju su scan i reduce funkcije za proizvoljne operacije.
```

### English
```
Write an algorithm for an EREW PRAM computer
that, for a given sequence of integer values of length n,
using at most n processors,
detects the length of the longest contiguous subsequence of elements with equal value
(print the length of the subsequence).
Scan and reduce functions for arbitrary operations are available.
```

**Requirement: O(n)**


## My solution

```
R[];

parallel (i = 0 ; i < n ; i++) { // O(n)
    R[i] = 1;
    x = P[i];
    for (j = i + 1 ; j < n ; j++) {
        if (x == P[j]) {
            R[i]++;
        } else {
            break;
        }
    }
}

print(max_reduce(R)) // O(log(n))
```
