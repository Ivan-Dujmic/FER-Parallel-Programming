## Task

### Croatian
```
Napisati algoritam za CRCW/EREW PRAM računalo
koji će za zadano polje P[]
odrediti broj različitih vrijednosti elemenata polja.
Npr. za polje [1, 2, 1, 3, 4, 2, 5, 1] rezultat iznosi 5.
Za polje od n elemenata na raspolaganju je n procesora.
Rezultat mora biti zapisan u jednoj izlaznoj varijabli.
```

### English
```
Write an algorithm for a CRCW/EREW PRAM computer
that will, for a given array P[],
determine the number of distinct values of the array elements.
For example, for the array [1, 2, 1, 3, 4, 2, 5, 1], the result is 5.
For an array of n elements, n processors are available.
The result must be written in one output variable.
```

**Requirement: O(n)**


## My solution

### CRCW
```
R[]; // Track if it has duplicates after itself

parallel (i = 0 ; i < n ; i++) { // O(1)
    R[i] = 1; // Assume it's unique initially   
}

for (i = 0 ; i < n - 1 ; i++) { // O(n)
    parallel (j = i + 1 ; j < n ; j++) {
        if (i + j < n) {
            if (P[j] == P[(j + i) % n]) {
                R[j] = 0; // It's not unique after all
                break;
            }
        }
    }
}

return add_reduce(R); // O(log(n))
```

### EREW
```
C[]; // We'll make a copy of P
R[]; // Track if it has duplicates after itself

parallel (i = 0 ; i < n ; i++) { // O(1)
    C[i] = P[i];
    R[i] = 1; // Assume it's unique initially
}

for (i = 1 ; i < n ; i++) { // O(n)
    parallel (j = 0 ; j < n ; j++) {
        if (i + j < n) {
            if (P[j] == C[(j + i) % n]) {
                R[j] = 0; // It's not unique after all
                break;
            }
        }
    }
}

return add_reduce(R); // O(log(n))
```
