## Task

### Croatian
```
Napišite algoritam složenosti log(n) za EREW PRAM računalo
koji će za zadano polje A[] s n elemenata
odrediti je li uzlazno sortirano.
Na raspolaganju je funkcija +_reduce(polje[])
koja provodi operaciju zbrajanja.
```

### English
```
Write an algorithm with complexity log(n) for an EREW PRAM computer
that will, for a given array A[] with n elements,
determine whether it is sorted in ascending order.
The function +_reduce(array[]) is available,
which performs the addition operation.
```

**Requirement: O(log(n))**


## My solution

```
R[];

parallel (i = 0 ; i < n - 1 ; i += 2) { // O(1)
    R[i] = (A[i] < A[i+1]) ? 0 : 1;
    if (i < n - 2) { // Avoid that one out-of-range access for even n
        R[i+1] = (A[i+1] < A[i+2]) ? 0 : 1;
    }
}

return add_reduce(R) == 0; // O(log(n))
```
