## Task

### Croatian
```
Napisati algoritam za EREW PRAM računalo složenosti manje od O(n)
koji će za dva ulazna niza znakova A[] i B[] duljine n
odrediti koji je prvi po abecedi
(program treba ispisati "A", "B" ili "jednaki").
Na raspolaganju je funkcija reduciranja
koja provodi proizvoljnu binarnu operaciju nad elementima polja.
```

### English
```
Write an algorithm for an EREW PRAM computer with complexity less than O(n)
that will, for two input character strings A[] and B[] of length n,
determine which one comes first alphabetically
(the program should output "A", "B", or "equal").
A reduction function is available
that performs an arbitrary binary operation over the elements of an array.
```

**Requirement: O(log(n))**


## My solution

```
// If 0 means that the elements at that location are the same
// Then we can just have a reduction where we do the following:
    // Return left if it's not 0
    // Otherwise return right
    // Then the leftmost non-0 will be propagated as a result
def cmp(num a, num b) -> num {
    return (a != 0) ? a : b;
}

R[];

parallel (i = 0 ; i < n ; i++) { // O(1)
    if (A[i] < B[i]) {
        R[i] = -1
    } else if (A[i] > B[i]) {
        R[i] = 1;
    } else {
        R[i] = 0;
    }
}

switch (cmp_reduce(R)) { // O(log(n))
    case -1:
        print("A");
        break;
    case 1:
        print("B");
        break;
    case 0:
        print("equal");
        break;
}
```
_Instead of doing if checks and returning {-1, 0, 1}, we could instead just subtract the ASCII character values._
