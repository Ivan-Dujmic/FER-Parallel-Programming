## Task

### Croatian
```
Napišite algoritam za EREW PRAM računalo složenosti manje od O(n^2)
koji će za matricu M[n][n]
odrediti predstavlja li permutacijsku matricu
(u svakom retku i stupcu je točno jedna jedinica, a ostali elementi su nule).
Program treba ispisati "da" ili "ne".
Na raspolaganju je n procesora
i funkcija reduciranja (O(log n))
koja provodi proizvoljnu binarnu operaciju nad elementima niza.
Pretpostavite da su elementi matrice bilo koje cjelobrojne vrijednosti
(mogu biti različiti od nula i jedan!),
te da se i-ti redak odnosno stupac matrice može dobiti kao R[i] odnosno S[i].
```

### English
```
Write an algorithm for an EREW PRAM computer with complexity less than O(n²)
that will, for a matrix M[n][n],
determine whether it represents a permutation matrix
(in every row and column there is exactly one one, and the remaining elements are zeros).
The program should output "yes" or "no".
n processors are available
and a reduction function (O(log n))
that performs an arbitrary binary operation over the elements of an array is available.
Assume that the matrix elements may be any integer values
(they may be different from zero and one!),
and that the i-th row or column of the matrix can be obtained as R[i] or S[i], respectively.
```

**Requirement: O(n \* log(n))**


## My solution

```
def check(num a, num b) -> num {
    if (a == 0 && b == 0) {
        return 0;
    } else if ((a == 0 && a == 1) || (a == 1 && b == 0)) {
        return 1;
    } else {
        // Otherwise they're either:
            // Both 1
            // At least one is neither 0 or 1
            // We've already returned -1 at some point (same as above)
        return -1;
    }
}

}
```
