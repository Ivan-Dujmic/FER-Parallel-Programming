## Task

### Croatian
```
Napisati algoritam za CRCW PRAM računalo koji će,
ispisivanjem "DA" ili "NE",
za zadano polje P[] sa n elemenata
odrediti predstavlja li permutacijski vektor
(permutacija skupa {1, 2, ..., n}).
Elementi polja mogu poprimiti samo cjelobrojne vrijednosti.
Primjerice, (3, 1, 2) je permutacijski vektor
dok (2, 4, 3), (1, 1, 2) i (4, 2, 1) nisu.
Na raspolaganju je funkcija reduciranja
koja provodi proizvoljnu binarnu operaciju nad elementima polja.
```

### English
```
Write an algorithm for a CRCW PRAM computer that will,
by printing "YES" or "NO",
for a given array P[] with n elements,
determine whether it represents a permutation vector
(a permutation of the set {1, 2, ..., n}).
The elements of the array may only take integer values.
For example, (3, 1, 2) is a permutation vector,
while (2, 4, 3), (1, 1, 2), and (4, 2, 1) are not.
A reduction function is available
that performs an arbitrary binary operation over the elements of the array.
```

**Requirement: O(n)**


## My solution

```
R[];
is_permutation = true;

parallel (i = 0 ; i < n ; i++) { // O(1)
    R[i] = 0;

    if (P[i] >= 1 && P[i] <= n) {
        R[P[i] - 1] = 1; // The vector has that element
    };

    if (R[i] == 0) {
        is_permutation = false;
    }
}

if (is_permutation) {
    print("YES");
} else {
    print("NO");
}
```
_This is O(1)._
