## Task

### Croatian
```
Napisati algoritam za EREW PRAM računalo
koji će za zadano polje P[] sa n elemenata
odrediti predstavlja li neki redak permutacijske matrice
(jedan element 1, svi ostali 0).
O vrijednostima elemenata polja se ništa ne pretpostavlja
(mogu biti bilo koje vrijednosti).
Na raspolaganju je funkcija *_reduce(polje[])
koja provodi proizvoljnu binarnu operaciju nad elementima polja.
```

### English
```
Write an algorithm for an EREW PRAM computer
that will, for a given array P[] with n elements,
determine whether it represents some row of a permutation matrix
(one element is 1, all the others are 0).
Nothing is assumed about the values of the array elements
(they may be any values).
The function *_reduce(array[]) is available,
which performs an arbitrary binary operation over the elements of the array.
```

**Requirement: O(log(n))**
