## Task

### Croatian
```
U ostvarenju igre 4 u nizu
prazni elementi ploče su označeni nulom,
a pozicije s igračevim žetonom jedinicom
(ne postoje elementi drugih vrijednosti).
Napišite algoritam za EREW PRAM računalo
koji za zadani (jednodimenzijski) niz elemenata
ploče P[] duljine n
otkriva postoji li u njemu 4 igračeva žetona u nizu
(ispisuje DA ili NE).
Na raspolaganju su scan i reduce funkcije za proizvoljne operacije.
```

### English
```
In the implementation of the Connect Four game,
empty board elements are marked with zero,
and positions with the player’s token are marked with one
(there are no elements with other values).
Write an algorithm for an EREW PRAM computer
that, for a given one-dimensional array of board elements
P[] of length n,
detects whether there are 4 of the player’s tokens in a row in it
(prints YES or NO).
Scan and reduce functions for arbitrary operations are available.
```

**Requirement: O(log(n))**


## My solution

```
R[];

parallel (i = 0 ; i < n - 3 ; i++) { // O(1)
    R[i] = P[i];
    for (j = 1 ; j <= 3 ; j++) {
        R[i] += P[i + j];
    }
}

if (max_reduce(R) >= 4) { // O(log(n))
    print("YES");
} else {
    print("NO");
}
```
